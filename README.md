# Portal GRR

Sistema web desenvolvido em Laravel para gerenciamento interno do Grupo de Resposta Rápida.

## Tecnologias utilizadas

- Laravel
- PHP
- Blade
- Bootstrap
- MySQL
- Git/GitHub

## Funcionalidades

- Login de usuários
- Painel administrativo
- Gestão de efetivo
- Relatórios operacionais
- Ranking
- Controle de permissões
- Módulo de RH

## Status do projeto

Projeto em desenvolvimento e evolução contínua.
